## 진행방식
1. 전주에 선정된 주제를 각자 마크다운 문서(개인블로그)로 정리해서 모두 발표
2. 면접 처럼 주제에 답변하고 필요시 꼬리 질문 에 답함
3. 해당 인원에 대한 피드백
4. 다음주 주제 다같이 정하기
5. 각자 받았던 질문 및 정리해서 깃에 업로드
